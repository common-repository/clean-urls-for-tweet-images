== Description ==

This plugin uses the post title in the URL for your tweet images posts instead of the default random hexadecimal identifier.  Tweet Images plugin required.


== Installation ==

1. Make sure you have installed and activated the <a href="http://wordpress.org/extend/plugins/tweet-images/">Tweet Images plugin</a>
1. Upload `tweet-images-clean-urls.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
